El Sistema

Sistema web para la gestión de competiciones.

Se suben los resultados de la semana.

El sistema actualiza clasificaciónes y realiza avisos.

  • Sanciones
  • Actualizaciones
  • etc.

El aspecto inovador

Actas electrónicas firmadas electronicamente.

Se substituyen las fichas por el DNI electrónico.

Tablets

Web continuamente actualizada, sin intervención de la federación

Funcionamiento

  1. El árbitro descarga los partidos del día
  2. Se crean, firman y almacenan las actas a medida que se juega
  3. Una vez en casa, el árbitro sube todas las actas

Financiación

Cada jugador invierte únicamente 6 € para la gestión de la competición.

Financiamos las tablets.

7 € mensuales durante 1 año, entonces pasa a ser propiedad de la federación.

Premiamos la fidelidad

+ años en el sistema -> - coste para sus jugadores

Cuantos más usuarios, más económico
